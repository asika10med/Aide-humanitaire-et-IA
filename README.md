# Aide-humanitaire-et-IA
IA améliore l’aide humanitaire en zones de crise en analysant automatiquement diverses données (satellites, réseaux sociaux, rapports). Elle permet d’identifier les besoins urgents, cartographier les risques et optimiser la distribution de l’aide. Le projet vise une réponse plus rapide, ciblée et efficace malgré les limites éthiques et techniques.
